# User Manual: Administrator Guide

## 1. Accessing Admin Mode
1.  Navigate to the homepage.
2.  Locate the **Invisible Button** at the bottom-right corner of the screen (or the "Login" link if visible in footer).
3.  Enter the default password: `admin`.
4.  You are now in **Edit Mode**. The `AdminBar` will appear at the bottom center.

## 2. Managing Content

### Adding a Project
1.  Scroll to the **Projects** section.
2.  Click the **+ (Plus)** card in the grid.
3.  Fill in the details (Title, Description, Tags, Link).
4.  Click **Save**. The grid updates immediately.

### Writing a Blog / Thought
1.  Navigate to **Second Brain**.
2.  Click **+ New Idea** in the Parking Lot columns.
3.  Type your thought.
    - *Public Column*: Visible to everyone.
    - *Private Column*: Visible only to you.
4.  To delete, hover over the card and click the **Trash Icon**.

## 3. Security Settings
1.  Click **SECURITY** in the Admin Bar.
2.  **Change Password**:
    - Enter current password.
    - **Biometric**: Click the scanner. If you have Windows Hello/Touch ID, follow the OS prompt.
    - **2FA**: If prompted, scan the QR code with Google Authenticator. Enter the 6-digit code.
    - Enter new password.
