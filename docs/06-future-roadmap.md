# Project Roadmap

## Phase 1: Current State (v1.0)
- [x] React Architecture
- [x] LocalStorage Persistence
- [x] Biometric Security
- [x] Second Brain UI

## Phase 2: Enhanced Connectivity (v1.1 - v1.5)
- [ ] **GitHub API Write Access**: allow the "Intellectual Changelog" to actually push commits to a real repo.
- [ ] **Spotify Integration**: "Listening Now" widget in the sidebar.
- [ ] **Export/Import**: JSON export of the entire "Second Brain" database for backup.

## Phase 3: True Backend (v2.0)
- [ ] **Supabase / Firebase Integration**: Move from LocalStorage to a real cloud DB to allow multi-device sync.
- [ ] **Server-Side Rendering (SSR)**: Port to Next.js for better SEO on individual blog posts.
- [ ] **PWA Support**: Offline capabilities and "Install to Home Screen".

## Phase 4: AI Integration (v2.5)
- [ ] **Local LLM**: Integrate `WebLLM` to have an AI assistant chat with your notes directly in the browser.
- [ ] **Auto-Tagging**: Use NLP to automatically tag new Ideas in the Parking Lot.
